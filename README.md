# BizOps

BizOps is a multi-tenant business management system that allows multiple companies to manage users, documents, and internal operations from a single platform with strict data isolation.

## Features
- Multi-tenant architecture
- Company and user management
- Role-based access control
- Activity logging
- Document management

## Tech Stack
- PHP (Vanilla)
- MySQL
- HTML, CSS, JavaScript
